# Shooting Game

以下、シューティングゲームを作成するに当たって必要な情報を記入する

## 実行例

敵の攻撃を受けると自分の体力が減る。自分の体力が0になったときゲームオーバーになる

![実行例1](GIF/image1.gif)

敵の体力を減らして0にするとゲームクリアになる

![実行例2](GIF/image2.gif)
